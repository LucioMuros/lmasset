# LMASSET

Sistema de gestão de ativos imobiliários.

## Tecnologias

React  
Vite  
TypeScript  
Supabase  
Recharts  
FullCalendar  

## Funcionalidades

Dashboard  
Gestão de propriedades  
Reservas  
Despesas  
Manutenção  
Relatórios  
Calendário  
Portal do investidor  

## Instalação

```bash
npm install

npm run dev
```
